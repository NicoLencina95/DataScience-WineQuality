## 📌 Resumen del Proyecto
Este repositorio contiene el desarrollo integral de un proyecto de Ciencia de Datos enfocado en el análisis y la predicción de la calidad del vino basándose en sus propiedades físico-químicas. 

El objetivo principal de este trabajo fue aplicar un enfoque analítico riguroso para entender qué factores químicos impactan positivamente en la evaluación sensorial de un vino, culminando en el entrenamiento de modelos de Machine Learning capaces de predecir su puntuación de calidad.

## 🎯 Objetivos del Análisis
1. **Análisis Exploratorio de Datos (EDA):** Evaluación de las distribuciones de variables químicas (acidez fija/volátil, ácido cítrico, azúcar residual, cloruros, pH, sulfatos, alcohol, etc.) y tratamiento de datos estructurados.
2. **Análisis de Correlación:** Identificación de las variables predictoras más fuertes (Feature Selection) que determinan si un vino es clasificado como de alta o baja calidad.
3. **Modelado Predictivo:** Ingeniería de características y entrenamiento de algoritmos de Machine Learning para estimar la calidad del vino con la mayor precisión posible.
